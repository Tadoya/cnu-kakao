# cnu-kakao
### 충남대학교의 학식, 실시간 도서관 좌석정보, 교내외 셔틀버스 노선정보를 확인 할 수 있는 봇(카카오톡 자동응답 API)

## 개발스펙
* django
* beautifulsoup4
* lxml
* heroku

## 기능
* 식당별 학식정보
* 실시간 도서관 좌석정보
* 학교버스 노선정보
